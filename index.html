📖 使用說明
🔗 網址新增
  • 點擊右上角「🔗 網址新增」按鈕開啟視窗
  • 貼入一至三個產品頁網址（第二、三個選填），點「開始搜尋」
  • 系統自動擷取品名、價格、材質、尺寸、圖片等資訊
  • 確認頁可直接修改任何欄位，空白表示網頁未提供，請手動填入
  • 多個網址會逐一確認，可選擇「略過」或「儲存，下一個」
＋ 手動新增
  • 點擊「＋ 手動新增」按鈕開啟表單
  • 可點擊或拖曳圖片區域上傳產品照片
  • 填寫品名（必填）、品牌、類型（必填）等欄位後按「儲存品項」
  • 只有叉叉或「取消」按鈕才會關閉視窗，點外部不會關閉
🔍 搜尋與篩選
  • 頂部搜尋欄可搜尋品名、品牌、材質、說明等文字
  • 點「篩選」按鈕（或從螢幕左緣向右滑）開啟分類側邊欄
  • 可依傢俱類型或品牌篩選，選完後自動收起
  • 右側排序選單可依價格、品牌、類別、年份等排序
🗂 傢俱分類
  • 預設分類：沙發、椅子、桌子、床架、層架、收納、燈具、地毯、戶外傢俱、裝飾品、其他
  • 新增品項時，在類型下拉選單選「＋ 自訂分類...」可輸入新分類名稱
  • 輸入名稱後按 Enter 即建立，自動選取該分類
📋 品項詳情與管理
  • 點擊任意卡片開啟詳情視窗，顯示所有欄位與來源網址
  • 有來源網址的品項可點「🖼 顯示產品照片」抓取原頁面最多 10 張照片
  • 詳情視窗內可點「✏️ 編輯」修改或「🗑 刪除」移除品項
  • 滑鼠移到卡片右上角也可直接編輯或刪除
🤖 Claude AI 搜尋備援
  • 網址擷取失敗時，若已設定 Anthropic API Key，系統會自動改用 Claude AI 搜尋
  • 在下方「API Key」欄位貼入金鑰即可啟用
  • API Key 存於本機瀏覽器，不會上傳至任何伺服器
